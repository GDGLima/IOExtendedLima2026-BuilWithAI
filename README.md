# IO Extended Lima 2026 - Build with AI

## 🚀 Acerca del Evento

**IO Extended Lima 2026 - Build with AI** fue un evento organizado por **GDG Lima (Google Developer Groups Lima)** enfocado en Inteligencia Artificial, agentes inteligentes y el ecosistema de Google Cloud.

Durante dos días, la comunidad exploró cómo construir soluciones modernas con **Gemini, Vertex AI, BigQuery, Google AI Studio y arquitecturas de datos impulsadas por IA**.
